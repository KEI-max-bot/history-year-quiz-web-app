# 年号道場

世界史の年号を効率的に覚えるためのクイズアプリ。

## 機能

- **入力モード**: 年号と出来事を登録して自分だけの問題集を作成
- **クイズモード**: カテゴリを選んで年号クイズに挑戦
- **Local Storage**: ユーザー追加データは自動保存

## 使い方

`app/index.html` をブラウザで開く

## 技術スタック

- HTML / CSS / JavaScript（バニラ）
- Local Storage（データ永続化）
