# Issue: 再読み込み時にクイズモードに戻される

## 概要
ページを再読み込みすると、入力モードで作業していてもクイズモードに戻されてしまう。

## 再現手順
1. アプリを開く
2. 入力モードに切り替える
3. ページを再読み込み（F5）する
4. クイズモードになっている

## 期待される動作
再読み込み前のモードを維持する、もしくは入力モードをデフォルトにする。

## 原因
`src/app.js` の起動処理で `switchMode("quiz")` がハードコードされている。

## 提案する修正
- Local Storageに現在のモードを保存して復元する
- または、起動時のデフォルトモードを設定可能にする

## 優先度
低（UX改善）
